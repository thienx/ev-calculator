# OpenClaw — EV Calculator

**Expected Value & Kelly Criterion Calculator for Crypto Trading**  
Live Binance prices • Built by Bobby

## Overview
OpenClaw EV Calculator is a powerful yet simple tool that helps traders quickly calculate:

- Expected Value (EV) of a trade
- Full Kelly & Quarter Kelly position sizing
- Optimal position size based on account capital
- Maximum profit and maximum loss

It supports leverage from 1x to 75x and fetches real-time prices from Binance via WebSocket.

## Features
- Live Binance price feed (BTC, ETH, SOL, etc.)
- Input: Entry Price • Take Profit • Stop Loss • Win Rate • Capital
- Automatic EV and Kelly calculations
- Clean, responsive UI (works great on mobile and desktop)
- Real-time reconnect button

## How to Use
1. Open the calculator: [https://thienx.github.io/ev-calculator1/](https://thienx.github.io/ev-calculator1/)
2. Enter your trade parameters
3. Click **Reconnect** if you need fresh prices
4. Check the **EV GAUGE** for results
